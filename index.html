Помощь

Приветствую вас на сайте программы F_Reference_H ! Эта программа создана в виде справочника, задача которого заключается в облегчении хранения данных и их систематизации.


В программе есть несколько вкладок:

  • Главная
  • Другое
    • Оптимизация ID
  • Настройки
  • Написать об ошибке
  • Помощь []

Главная

Эта вкладка основная. На ней вы можете добавить новую запись [], удалить (предварительно выделив нужную) [], обновить содержимое списков записей [], а двойным щелчком по записи вы можете её открыть для просмотра и редактирования.

Добавить новую запись. Это сделать очень просто! Достаточно нажать на соответствующую кнопку [] над тем списком, в который вы хотите добавить новую запись. списком записей.

Удалить запись. Просто нажмите один раз на запись, которую хотите удалить и над списком, в котором вы сделали выделение, нажмите на кнопку удаления записи []

Обновить содержимое. Эта кнопка [] служит для обновления обоих списков записей.

Открыть/Редактировать запись. Это сделать достаточно просто. Просто дважды быстро нажмите на интересующую вас запись и она откроется вам для редактирования и просмотра.
